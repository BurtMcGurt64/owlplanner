# UI/Frontend Plan (React)

## Version 1
* Single page containing the following functions
    * User selects/types in courses
    * User sets preferences
    * User clicks "generate schedules"
    * Display the generated schedules
* Components:
    * Header w/ "OwlPlanner" as the title
    * CourseSearch for a search bar
    * SelectedCourses, small 'chips' for showing which courses have been selected
    * PreferencePanel for checking preferences
    * ScheduleResults for a list of schedules
* General Layout:
    * OwlPlanner banner at top
